⏳ Gestión de Tiempo y Horas de Trabajo
📱 Aplicación Android desarrollada en Kotlin con Android Studio, integrada con Firebase para gestionar el tiempo de trabajo, registrar horas y analizar la productividad.

🚀 Características Principales
✅ Registro y control de horas trabajadas
✅ Inicio y fin de jornada con un solo clic
✅ Almacenamiento seguro en Firebase Firestore
✅ Autenticación con Firebase Authentication
✅ Sincronización en tiempo real entre dispositivos
✅ Estadísticas y reportes de productividad
✅ Diseño UI intuitivo basado en Material Design

🛠 Tecnologías Utilizadas
Lenguaje: Kotlin
IDE: Android Studio
Base de datos: Firebase Firestore
Autenticación: Firebase Authentication
Interfaz gráfica: Material Design
Arquitectura: MVVM (Model-View-ViewModel)
📦 Instalación y Configuración
1️⃣ Clonar el repositorio
bash
Copy
Edit
git clone https://github.com/tu_usuario/tu_repositorio.git
cd tu_repositorio
2️⃣ Abrir en Android Studio
Abrir Android Studio
Seleccionar "Open an existing project"
Navegar hasta la carpeta clonada y seleccionar el proyecto
3️⃣ Configurar Firebase
Crear un proyecto en Firebase en Firebase Console
Agregar una aplicación Android usando el ID del paquete
Descargar el archivo google-services.json y colocarlo en app/
Activar Firestore Database y Authentication en Firebase
4️⃣ Ejecutar la aplicación
Conectar un dispositivo o emulador
Presionar Run ▶ en Android Studio
